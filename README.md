# Калькулятор страховых взносов

Данный проект предназначен для расчета страховых взносов за определенный период (начало и конец отрезка времени), с возможностью определения суммы взносов для каждого квартала.

## Структура проекта

- `calc.py` - основной класс `Calc`, который выполняет все необходимые расчеты
- `main.py` - пример использования класса `Calc`

## Функциональность

### Класс `Calc`

Класс `Calc` предназначен для расчета страховых взносов за заданный период. В классе определены:

- `quarter` - словарь с датами начала и конца каждого квартала
- `year` - словарь с заданными значениями страховых взносов для каждого года (в данном случае 2023)

#### Параметры

- `start_date` - начальная дата периода (в формате `DD.MM.YY`)
- `end_date` - конечная дата периода (в формате `DD.MM.YY`)

#### Методы

- `calculation(item)` - рассчитывает сумму взносов для заданного значения `item` за указанный период
- `get_contribution()` - возвращает словарь со значениями страховых взносов для года начальной даты
- `get_format_to_date(data)` - преобразует строку даты в формат `datetime`
- `get_to_quarter(key, value)` - возвращает список строк с суммами взносов для каждого квартала

### Пример использования

```python
from calc import Calc

start_date = '01.01.23'
end_date = '31.08.23'

cl = Calc(start_date, end_date)
tax_type = cl.get_contribution()
res = {key: cl.calculation(value) for key, value in tax_type.items()}
res1 = [cl.get_to_quarter(key, value) for key, value in res.items()]
[print(*i, sep='\n') for i in res1]
print(*res.items(), sep='\n')
print('Итог:', sum(res.values()))
```

## Установка и запуск

Для запуска проекта достаточно запустить файл `main.py`:

```
python main.py
```

## Вывод

Программа выводит суммы страховых взносов для каждого квартала и итоговую сумму за весь период.
